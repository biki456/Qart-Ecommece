The application allows users to browse products, manage shopping carts, and complete purchases using UPI payment.

---

## 📌 Project Description

QART is a simple e-commerce platform designed to demonstrate full-stack web development concepts. The application includes product listing, user authentication, shopping cart functionality, and a basic checkout system using UPI QR code payments.

This project showcases backend development using **Django**, database management, and frontend UI integration.

---

## 🚀 Features

- User registration and login system
- Product listing and product detail pages
- Product search functionality
- Add to cart and cart management
- Checkout system
- UPI QR code payment
- Responsive UI design
- Admin panel for managing products

### 2️⃣ Navigate to project folder
cd Qart-Ecommece


### 3️⃣ Install dependencies
